# customer_churn_analysis
This project analyzes customer churn using a telecom dataset of 7,043 customers to identify the key drivers behind customer attrition. Data preprocessing included handling missing values, converting TotalCharges to numeric format, and standardizing categorical variables before performing exploratory data analysis and visualization.

**Value-Added Services**

Customers without services like Online Security, Tech Support, and Device Protection have churn rates close to 40%+, nearly double compared to users of these services.

**Payment Method**

Electronic Check users: ~45% churn

Auto-pay methods: below 20% churn
